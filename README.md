## ğŸš€ Quick Start

### Prerequisites
- Python 3.8 or higher
- pip package manager
- Git

### Installation

1. **Clone the repository**:
```bash
git clone https://github.com/HavilahAcademy/PlotSenseAI-Hackathon-Demo-Projects.git
cd PlotSenseAI-Hackathon-Demo-Projects
```

2. **Create a virtual environment** (recommended):
```bash
python -m venv plotsense-env
source plotsense-env/bin/activate  # On Windows: plotsense-env\Scripts\activate
```

3. **Install PlotSenseAI**:
```bash
pip install plotsense
```

4. **Choose your project and follow its specific setup guide**:
   - [Project One Setup](./project_one/README.md#setup-instructions)
   - [Project Two Setup](./project_two/README.md#cloning-and-installation)
   - [Project Three Setup](./project_three/README.md#installation--setup)
